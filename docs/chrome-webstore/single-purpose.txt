This extension has one purpose: to prevent duplicate browser tabs. When you open a tab with a URL that is already open, the extension detects it and closes or switches tabs according to your chosen setting (e.g. close the new tab, or close the old one and switch). You can choose to check duplicates across all windows or only the current window. When switching to a tab in another window, the extension focuses that window; if removing a duplicate leaves a window empty, it closes that window. You can prevent duplicates on all websites except listed exceptions, or only on sites you add to a monitored list. Per-site rules override global defaults where configured. You can also open a single tab without prevention via the right-click menu “Open in new tab (allow duplicate).” All behavior is limited to tab and window management for duplicate prevention; no other functionality is provided.
